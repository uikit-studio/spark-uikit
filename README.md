# Spark

A bold marketing UI kit — big type, orange, gradient accents. Generated with
[`uikit-standard`](https://github.com/uikit-studio) for the UIKit gallery.

➡️ **[USAGE.md](./USAGE.md)** · manifest: [`uikit.json`](./uikit.json)

```bash
git clone https://github.com/uikit-studio/spark-uikit my-app
cd my-app && npx uikit-studio init
```
